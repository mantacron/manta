What's included

Everything you need to ship clean code.
Nothing you have to configure.

11 agents, 17 commands — security, code generation, quality enforcement, and docs automation all in one install.

Security on every commit

OWASP Top 10, hardcoded secrets, API keys, SQL injection, XSS, and auth bypass — all caught and blocked before they reach your repo. No configuration needed.

Complete implementations

/project:write "feature" generates production-ready code with rate limiting, auth, validation, pagination, and tests baked in. No TODOs, no stubs.

Quality & performance

DRY violations, cyclomatic complexity, N+1 queries, memory leaks, and unsafe database migrations are caught before they compound into something harder to fix.

Docs that stay current

/project:update-docs keeps README and CHANGELOG in sync with your commits. PR summaries generated automatically after every push.
